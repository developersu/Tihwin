Tihwin adalah alat yang kompatibel dengan **OPL** untuk membuat berkas permainan yang terbagi menjadi beberapa bagian (fragmented).  
Mirip dengan **"ul.cfg manager"** atau **"USBUtil"**, dan mendukung sistem operasi **Mac** serta **Linux**.
